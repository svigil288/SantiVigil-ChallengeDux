# SantiVigil-ChallengeDux

## Challenge Description

This project is a web application developed as part of the technical challenge for Dux Software. The application allows managing a user list with complete CRUD functionalities and advanced filtering capabilities.

## Technologies Used

- **Frontend**: Next.js with TypeScript
- **API**: Dux Software API
- **Styling**: CSS Modules / Prime React
- **State Management**: React Hooks
- **HTTP Client**: Axios

## Features

- ✅ Create new users via reusable modal form
- ✅ Edit existing users using the same modal component
- ✅ Delete users
- ✅ Search users by name or surname (partial match)
- ✅ Filter users by status (active/inactive)
- ✅ Remove filters
- ✅ Paginated records using `_limit` & `_page` parameters
- ✅ Persistent state management
- ✅ Responsive design

## Installation and Local Execution

### Prerequisites

- Node.js (v18.17 or higher)
- npm or yarn
- Git

### Steps to run locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/SantiVigil-ChallengeDux.git
   cd SantiVigil-ChallengeDux
   cd client
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the application**
   ```bash
   npm run dev
   ```

4. **Access the application**
   - Application: http://localhost:3000
